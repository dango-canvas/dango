### **1. 总体与环境规格**

**1.1. 应用程序形态**
*   **双重形态**: 本质是一个单页 Web 应用 (SPA)，既可以作为独立网站运行，也打包为浏览器扩展。
*   **单文件构建 (build.ts)**: 提供基于 Bun 的构建脚本，将 CSS/JS/资产内联为单一 HTML，支持混淆与脚本标签转义，确保零依赖分发。
*   **浏览器扩展行为**:
    *   扩展点击逻辑 (`background.js`): 点击扩展图标时，会检测当前浏览器窗口是否已有打开的 Dango 标签页。如有则跳转并高亮该标签页，否则新建标签页，确保“单例”体验。
    *   兼容性: 同时支持 Chrome (Manifest V3) 和 Firefox (Manifest V2)，通过 `background.js` 中的 API 抽象层实现。

**1.2. 国际化 (i18n)**
*   **语言支持**: 支持简体中文 (`zh`) 和 英文 (`en`)。
*   **检测与持久化**: 优先读取 `localStorage` (`cc-lang`)；若无，则检测浏览器语言 `navigator.language`；默认回退为英文。
*   **动态更新**: 切换语言时，实时更新页面标题 (`document.title`)、DOM 文本 (`data-i18n`)、悬浮提示 (`title`) 和 输入框占位符 (`placeholder`)。
*   **资源国际化**: 背景图片 URL 支持国际化配置。若用户未自定义背景，系统将根据当前语言自动应用 i18n 预设的默认底图，确保跨语言环境下的视觉氛围一致性。

**1.3. 主题系统**
*   **模式**: 提供 🌞 浅色 (Light)、🌙 深色 (Dark) 与 🌓 自动 (Auto) 三种模式。
*   **实现**: 基于 CSS 变量 (`:root` vs `[data-theme="dark"]`)。
*   **自动模式**: 跟随系统 `prefers-color-scheme`，系统主题变化时实时刷新。
*   **持久化**: 设置保存在 `localStorage` (`cc-theme`)。

### **2. 画布 (Canvas) 核心规格**

**2.1. 视图交互**
*   **无限画布**: 支持无限平移和缩放。
*   **平移 (Pan)**:
    *   **鼠标**: 按住空格键 + 拖拽，或按下鼠标中键 + 拖拽。
    *   **触控**: 单指在空白处拖拽。
    *   **性能优化**: 平移操作仅更新容器 CSS `transform` 属性，跳过节点重绘，实现 60fps 流畅体验。
*   **缩放 (Zoom)**:
    *   **鼠标**: `Ctrl/Cmd` + 滚轮，或 `Ctrl` + `+`/`-`/`=` 键。
    *   **触控**: 双指捏合 (Pinch)，支持中心点缩放计算。
    *   **性能优化**: 缩放操作同样采用 CSS `transform` 更新，并在操作结束时防抖保存数据。
*   **视图重置**: 按 `Home` 键或 `Ctrl` + `0`，视图平滑动画回归中心 (`scale: 1.2`)。
*   **快速全局预览 (Quick Global Navigation)**:
    *   **触发机制**: 快速双击鼠标中键（间隔 < 300ms），并在第二次点击时保持按住状态。
    *   **进入逻辑**: 立即触发快速 `fitView` 动画（200ms），使画布所有内容进入视野，进入“全局视角”。
    *   **交互反馈**: 按住期间，光标变为抓取状态 (`grabbing`)。用户可以自由移动鼠标选择目标着陆区域。
    *   **释放跳转**: 松开中键时，视图执行平滑动画（500ms）移动至鼠标当前指向的世界坐标，并自动放大。
    *   **智能缩放**: 跳转后恢复至进入全局模式前的缩放级别；若原级别过小，则自动补正至 `0.8`，确保着陆后内容清晰易读。

**2.2. 背景与视觉反馈**
*   **网格**: 默认显示点状网格背景 (`radial-gradient`)。可设置隐藏。
*   **空状态引导**: 当画布无节点时，显示手绘风格的箭头和提示文字（“输入想法，开启你的画布”），引导用户关注输入框。
*   **拖拽反馈**: 当用户拖拽文件进入画布区域时，画布边缘显示虚线框高亮 (`.drag-over`)，提示可释放导入。

### **3. UI 组件规格**

**3.1. 主控制面板 (`#ui-layer`)**
*   **形态**: 位于左上角的悬浮面板。默认收起为方形图标，鼠标悬浮 (`hover`) 或 内部聚焦 (`focus-within`) 时展开。
*   **移动端适配**: 移动端点击面板区域时强制展开 (`.mobile-active`)，点击画布区域时自动收起。
*   **输入区 (`textarea`)**:
    *   支持批量输入。
    *   **解析规则**: 
        *   **智能拆分**: 支持换行符和逗号（中英文逗号均可）拆分节点。
        *   **引用保护**: 识别中英文双引号/单引号 (`"`, `'`, `“`, `‘`)，引号内的内容（包括其中的逗号和换行）会被视为单个节点处理。
    *   **布局规则**: 新节点组放置在现有节点包围盒最右侧 `x + 200px` 的空白区域起点。
    *   **快捷键**: `Ctrl/Cmd + Enter` 立即生成节点，`Enter` 仅换行。
*   **操作堆栈 (`.action-stack`)**:
    *   采用 3D 翻转动画结构。
    *   **正面**: [导出] [导入] [清空]。
    *   **反面 (点击导出后)**: [LINK] [DANGO] [EMBED]。
    *   **自动复位**: 导出面板翻转后，若无操作或点击外部，会自动翻转回正面。

**3.2. 顶部状态栏**
*   **Logo**: 根据日期自动显示节日 Emoji（如 🧧春节, 🎃万圣节, 🎄圣诞节等），平时显示 ✨。
*   **功能区**: 语言切换、主题切换、设置、帮助。

*   **模态框与弹窗**
*   **搜索功能 (`#search-container`)**:
    *   **形态**: 位于右上角的悬浮面板（`top: 27px`, `right: 24px`），确保不干扰左侧主菜单且符合操作习惯。
    *   **交互逻辑**:
        *   **唤起**: 通过 `Ctrl + F` 快速开启，焦点自动定位至输入框并全选已有内容。
        *   **实时检索**: 支持关键词实时过滤，不区分大小写。
        *   **搜索进度**: 动态显示匹配结果数量（如 `1/5`）。
        *   **导航**: 提供“上一个/下一个”按钮及快捷键（`Enter` / `Shift + Enter`）循环切换。
        *   **关闭**: 点击关闭按钮或按 `Esc` 键退出，同时清除所有搜索相关状态。
    *   **视觉反馈**:
        *   **自动定位**: 搜索命中后，画布执行平滑动画将目标节点居中展示，并自动维持合适的缩放比例。
        *   **高亮动画**: 命中的节点会应用 `ping-highlight` 呼吸灯动画（基于当前主题色），帮助用户在复杂画布中瞬间锁定目标。
*   **设置面板 (`#settings-modal`)**:
    *   隐藏网格点。
    *   Alt 键兼任 Ctrl 功能 (方便单手操作)。
    *   **画布底图**: 
        *   允许输入任意 HTTP/HTTPS 图片 URL。图片会以 `cover` 模式挂载在最底层 (`html` 标签)，并在上方 (`body` 标签) 覆盖一层无模糊的半透明灰色遮罩 (`rgba(127, 127, 127, 0.3)`)，完美保留网格线并确保节点易读性。
        *   **交互优化**: 输入框占位符通过 `↵` 引导用户按回车生效。在输入框内按 `Enter` 后，光标将自动失焦（移除闪烁）以应用设置，同时保持面板开启状态，确保操作连贯。
    *   **持久化与同步**: 该配置作为“画布氛围”的一部分，不仅持久化于本地，还会全量导出至 JSON 文件和分享链接中，确保不同终端下的视觉体验完全一致。
    *   手写风格 (实验性)。
*   **帮助面板 (`#help-modal`)**: 列出快捷键，提供 GitHub/反馈/关于 入口。
*   **关于弹窗 (`#about-card`)**: 包含版本信息、简介、GitHub Star 引导、博客链接。支持点击遮罩层关闭。
*   **Toast 通知**: 底部浮出提示。对于“清空”、“导入”等破坏性操作，提供 **[撤销]** 和 **[导出备份]** 按钮作为安全网。

### **4. 画布元素规格**

**4.1. 节点 (Node)**
*   **渲染逻辑**:
    *   **变更检测 (Change Detection)**: 通过 `data-lastText` 属性对比节点文本，只有内容真正变化时才重新执行 Markdown 解析和 DOM 更新，大幅降低重绘开销。
    *   **普通文本**: 支持 Markdown 基础语法（粗体 `**`、斜体 `*`）。
    *   **标题节点 (Heading)**: 识别 `#`, `##`, `###` 前缀。
        *   **样式**: 应用 `.node-h1/2/3` 类，实现阶梯式字号（24/20/17px）与内边距缩放。
    *   **注释节点 (Comment)**: 识别 `//` 前缀。
        *   **功能定位**: 专为连线旁的语义标注设计（如实体关系建模）。
        *   **样式**: 应用 `.node-comment` 类，无边框、无背景、无投影，使用 12px 浅色文字，使其在视觉上不抢占重心。
    *   **代码卡片 (Code Card)**: 识别 ` ``` ` 包裹的块。
        *   **样式**: 深色拟物化卡片 (`#1e293b`)，左上角带有 macOS 窗口风格的红黄绿圆点装饰。
        *   **解析逻辑**: 自动提取首行语言标签并展示在右上角，内部通过纯正则实现轻量的语法高亮（涵盖关键词、字符串、注释等）。
    *   **行内链接 (Inline Link)**: 识别 `[text](url)`。
        *   **样式**: 极致的色彩融合。彩色节点内继承当前文字颜色；白色节点内应用品牌色，配以半透明下划线。
        *   **交互**: 鼠标悬浮时带有半透明高亮；点击链接时通过阻断事件冒泡，在新窗口中打开而不触发节点自身的选中/拖拽。
    *   **多行支持**: 包含换行符的节点会自动应用 `.has-multiline` 样式。
    *   **色彩系统**: 支持 9 种颜色（`c-white`, `c-red`, `c-yellow` 等）。可通过 `Alt + 1-9` 快速切换选中节点的颜色。
    *   **Todo 列表**: 识别 `[ ]` 或 `[x]`。渲染为带复选框的列表。
        *   **交互**: 点击复选框区域 (`.todo-checkbox-wrapper`) 可直接切换完成状态（修改原始文本），并触发重绘。
        *   **样式**: 包含 Todo 的节点强制左对齐。
    *   **链接节点**: 识别 URL。渲染为带下划线的文本，选中节点后右侧悬浮“跳转按钮”。
    *   **图片节点**: 内容完全匹配 Markdown 图片格式 `![alt](url)` 时渲染为图片。
        *   **初始尺寸**: 默认小尺寸 (宽 100px，按原始比例计算高度)。
        *   **尺寸切换**: 选中图片节点时显示尺寸按钮，图标会根据下一级尺寸动态变化（S → L）。点击按 S(100) → L(200) 循环。
        *   **数据同步**: 尺寸变化仅更新节点已有 `w`/`h` 字段。
*   **编辑**:
    *   双击节点或按 `Enter` 进入编辑模式 (`contentEditable`)。
    *   编辑时暂时移除 Markdown 渲染，显示原始文本。
    *   `Shift + Enter` 换行，`Enter` 完成编辑，`Esc` 取消。
*   **手写风格**: 开启设置后，应用 `Architects Daughter` (英) / `LXGW WenKai` (中) 字体，并应用不规则边框圆角。

**4.2. 连接线 (Link)**
*   **渲染**: SVG `<line>` 元素。
    *   **DOM 复用 (Recycling)**: 采用 Diffing 机制，复用已存在的 `<line>` 节点，仅更新坐标属性 (`x1/y1`)，避免频繁的 DOM 创建与销毁。
    *   **箭头渲染**: 使用 SVG `<marker>` 实现端点箭头。
*   **交点计算**: 使用 `getEdgeIntersection` 算法，确保连线精确指向节点矩形的边缘，而非中心点。
*   **方向控制**: 支持三种状态：无方向、指向目标 (Target)、指向源头 (Source)。
*   **样式**: 虚化处理 (`opacity: 0.35`)，箭头为 V 形描边而非填充三角形，更具手绘感。

### **5. 核心交互与操作**

**5.1. 历史记录 (Undo/Redo)**
*   **机制**: 基于快照 (Snapshot) 的历史栈。
*   **容量**: 最大 50 步。
*   **触发**: 节点增删改、移动、连线、属性变更、选中状态变化均记录历史。
*   **撤销还原**: 历史快照包含当前选中状态 (`selection`)，撤销时可同步恢复选中。
*   **快捷键**: `Ctrl+Z` (撤销), `Ctrl+Y` 或 `Ctrl+Shift+Z` (重做)。

**5.2. 鼠标/触控操作**
*   **选择**: 单击选中；拖拽框选；`Ctrl` + 单击 (多选/反选)；`Ctrl+A` (全选)。
*   **移动**: 拖拽选中节点。
*   **复制/粘贴**: `Ctrl+C` 复制节点数据到内部剪贴板；`Ctrl+V` 粘贴。粘贴时节点会带有 `20px` 的偏移。
    *   **系统集成**: 复制时同步将节点纯文本写入系统剪贴板。
*   **克隆**: 按住 `Ctrl` (或 `Alt` 若开启设置) 拖拽节点，会在原位置保留副本，拖走新生成的节点（原地克隆逻辑）。
*   **双击**:
    *   节点上双击进入编辑。
    *   空白区域双击在指针位置创建新节点并自动进入编辑。
*   **中键双击并按住**: 开启“全局视角”导航。进入时全景缩放，松开时根据鼠标落点精准降落并平滑放大。
*   **触控特化**:
    *   **轻触**: 选中节点/组。
    *   **拖拽**: 移动节点/平移画布。
    *   **双指捏合**: 缩放画布。
    *   **拟态双击**: 短时间内两次轻触同一节点触发编辑。

**5.3. 快捷键体系**
*   **基础**: `Del` (删除), `Space` (平移), `Home` (归位)。
*   **搜索**: `Ctrl+F` (开启搜索), `Enter` (下一个结果), `Shift+Enter` (上一个结果), `Esc` (关闭搜索)。
*   **组合**: `Ctrl+G` (编组), `Ctrl+Shift+G` (解组)。
*   **导出 (`Ctrl+S`)**: 快速触发 JSON 文件导出。
*   **连线 (`Ctrl+L`)**: 状态机循环：
    1.  无 -> 建立连线 (无方向)。
    2.  无方向 -> 指向目标 (Target)。
    3.  指向目标 -> 指向源头 (Source)。
    4.  指向源头 -> 删除连线。
*   **快捷生成 (Ctrl + 方向键)**:
    *   **触发条件**: 仅当选中 1 个节点时生效。
    *   **交互过程**: 
        *   按下 `Ctrl` + `方向键` 时，在对应方向（距离 80px）显示虚线预览节点及指向预览节点的连线（不会因为长按连续生成）。
        *   **连线模式切换**: 在预览状态下，再次按下相同的方向键可循环切换连线模式：**指向目标 (Target)** -> **无箭头 (None)** -> **无连线 (Detached)**。切换后的模式在当前预览过程中（如切换不同方向键时）会得到保留。
        *   松开按键时，真正生成新节点，并根据选定的模式建立连线，最后自动选中新节点并进入编辑状态。
    *   **设置联动**: 若开启了 "Alt 键兼任 Ctrl 功能"，按 `Alt + 方向键` 也能触发（注意：选中多个节点时，该快捷键将执行原有的方向对齐功能，两者互不冲突）。
*   **微移 (Nudge)**: 使用 `方向键` (无修饰键) 对选中节点进行 10px 级别的微调。
*   **色彩**: `Alt + 1~9` 快速切换选中节点的背景颜色。
*   **对齐**:
    *   `Alt` + `W/A/S/D` 或 `方向键` (顶/左/底/右对齐)。
    *   `Alt` + `H/J` (水平/垂直居中)。
    *   `Alt` + `Shift` + `H/J` (水平/垂直分布)。
    *   **智能对齐 (`Alt + .`)**: 
        *   **多模式识别**: 自动根据节点分布识别为“网格”、“横向”或“纵向”趋势。
        *   **动画过渡**: 使用 `animateNodesTo` 执行平滑动画，并保持整体重心位置不变。
*   **聚光灯 (`Q`)**: 按住 Q 键压暗背景，高亮鼠标位置。

### **6. 数据与输入输出**

**6.1. 数据存储**
*   **本地**: `localStorage` (`cc-canvas-data`) 实时自动保存。
*   **数据结构**: 包含 `nodes`, `groups`, `links` 数组以及 **视图状态** (`view`: `{x, y, scale}`)。
    *   **视图恢复**: 刷新或重新打开页面时，会自动恢复上次的视点位置和缩放级别。

**6.2. 导出**
*   **JSON (.dango)**: 完整数据备份，包含节点、连线、分组及**全局设置**。支持快捷键 `Ctrl+S`。
*   **链接 (Link)**:
    *   使用 `LZString` 压缩当前数据为 URL Hash。
    *   采用紧凑型数组结构，大幅缩短分享链接长度。
    *   复制分享链接到剪贴板。
*   **嵌入代码 (Embed)**:
    *   生成 `<iframe>` 嵌入代码块并复制到剪贴板。

**6.3. 导入**
*   **文件**: 支持点击按钮选择文件，或直接拖拽 `.dango` 文件到画布。
    *   **视图重置**: 导入文件时，视点将强制重置为默认状态（中心位置，缩放 1.2）。
*   **URL 加载**: 打开页面时若 URL Hash 存在数据，优先解压加载 Hash 数据（分享预览模式）。
    *   **视图重置**: 通过链接打开时，视点同样强制重置为默认状态。

### **7. 特殊模式**

**7.1. 嵌入模式 (Embed Mode)**
*   **触发**: URL 参数 `?embed=true`。
*   **UI 变化**: 隐藏左上角主控面板。右下角显示“预览信息”和“打开完整版”按钮。
*   **行为**: 允许用户交互（拖拽、编辑），但**不会**将修改保存到 `localStorage`，刷新即恢复原状。适用于博客/文档嵌入演示。
*   **适配优化**:
    *   **自动缩放**: 嵌入模式下初始缩放比例设为 `1`，并自动执行 `fitView` 以适应较窄的容器（如 Obsidian）。
    *   **动态跳转**: “打开完整版”按钮链接会随画布内容的修改实时更新。
    *   **安全降级**: 针对 Obsidian 等沙盒环境限制 `window.open` 的问题，实现二级降级：
